Hotstreak
=========

Hotstreak is a "seinfeld"-calendar. See http://lifehacker.com/281626/jerry-seinfelds-productivity-secret .

Uses:
- Django:                       http://www.djangoproject.com/
- Django-Tastypie:              http://tastypieapi.org/
- Jquery:                       http://jquery.com/
- Jquery UI:                    http://jqueryui.com/
- Backbone.js                   http://documentcloud.github.com/backbone/
- Underscore.js                 http://documentcloud.github.com/underscore/
- Parts of backbone-tastypie:   http://github.com/PaulUithol/backbone-tastypie
- Twitter Bootstrap             http://twitter.github.com/bootstrap/

Hosted on Heroku http://www.heroku.com/
